Get started with Elfsight Form Builder 1.0.1

Installation

1. Install Elfsight Form Builder WordPress Plugin elfsight-form-builder-cc.zip to your website. Don't know how? Please learn here.

2. Go to Form Builder page in WordPress admin menu.

3. Create a new widget, adjust all settings and get a ready-made shortcode.

Use Form Builder as a WordPress Shortcode



1. Create a new widget

You can use Elfsight Form Builder as a WordPress shortcode. To do this, you have to create a new widget, set all options in the Elfsight editor and save it.

2. Copy-paste the shortcode

You will see the new widget's shortcode in the widgets list. You can copy-paste it into the required place of your website.

Example of the Elfsight Form Builder shortcode:

[elfsight_form_builder id="1"]

Use Form Builder as a
Gutenberg Block

You can use Elfsight Form Builder as a block for Gutenberg editor. Just add it to your page during page building and select one of the available Form Builder widgets from the list.

Use Form Builder as a
WordPress Widget

To use Elfsight Form Builder as a WordPress widget, go to Appearance -> Widgets, and drag Elfsight Form Builder WordPress widget to any widget section. Next, select one of available Form Builder widgets from the list.

Use Form Builder as a
Visual Composer Element

You can use Elfsight Form Builder as a Visual Composer element. Just add it to your page during page building and select one of the available Form Builder widgets from the list.
